# 🚀 Errand Matching App 🚀  
### 본교 대학생 대상 심부름 매칭 어플  
🚧 *프로젝트 진행중* 🚧  
&nbsp;

## Description
- 본교인 안동대학교 학생을 대상으로 한 심부름 앱을 제작하고, 배포하고자 함  
- 에브리타임에서 학생들끼리 심부름을 요청하고 요청받는 게시글이 빈번하게 보여 생각한 프로젝트 주제  
- 안동대 학생 이메일로 회원가입을 하고, 이메일 인증까지 완료해야 앱을 이용할 수 있음  
- ...
